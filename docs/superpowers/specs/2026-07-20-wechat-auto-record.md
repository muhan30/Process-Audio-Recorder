# SPEC: 微信通话自动检测 + 自动录音

> 版本: 1.0 | 日期: 2026-07-20 | 状态: 待审查

---

## 一、背景

用户每次录微信通话需手动操作：打开 APP → 等微信出 PID → 选中 → 点开始录音。参考 WeChatRecorder.py 的实现，微信通话时桌面上会出现特定类名的窗口，可以通过检测这些窗口来全自动触发录音。

## 二、需求

### 需求 1：微信通话窗口检测

**目标**：实时检测微信是否在通话中，无需轮询音频会话。

**原理**：微信通话时会创建以下窗口（不可见，但仍存在于桌面窗口列表中）：

| 窗口类名 | 含义 |
|---------|------|
| `ILinkAudioWnd` | 语音通话窗口 |
| `AudioWnd` | 音频控制窗口 |
| `ILinkVoipTrayWnd` | 通话悬浮窗 |

这三个窗口同时存在 = 微信正在通话。所有窗口消失 = 通话结束。

**验收标准**：
- 每 1 秒枚举所有顶层窗口，检查类名
- 三个窗口至少存在一个 → 判定"微信在通话"
- 全部消失 → 判定"通话结束"
- 误判率 ≈ 0（这三个类名是微信专属，其他软件不会用）

### 需求 2：自动开始/停止录音

**目标**：检测到微信通话 → 自动开始录音。通话结束 → 自动停止。

**流程**：

```
检测到通话窗口出现
        │
        ▼
  在会话列表中找 WeChat.exe / Weixin.exe
        │
   ┌────┴────┐
   │ 找到了？  │
   └────┬────┘
   是   │   否
   ▼    │   ▼
 自动   │  每1秒重试，
 开始   │  持续10秒仍未找到→放弃
 录音   │
        ▼
  监听窗口，全部消失？
        │
   是   │   否
   ▼    │   ▼
 自动   │  继续监听
 停止   │
 录音
```

**验收标准**：
- 通话开始后 2 秒内自动开始录音（需等会话列表刷新出微信 PID）
- 通话结束后 3 秒内自动停止（防止窗口闪烁导致的误停）
- 当前已在录其他软件时，不触发自动录音（尊重用户手动操作）
- 自动录音使用当前的设置配置（增益、滤波、噪声门等）
- 录音文件命名不变：`recordings/微信/日期 微信 时间.m4a`

### 需求 3：自动模式开关

**目标**：用户可选择是否启用自动模式。

**验收标准**：
- 主窗口待机态新增一个复选框："微信通话自动录音"
- 默认勾选（首次运行）
- 取消勾选 → 完全不做窗口检测，恢复手动模式
- 状态写入 INI：`[Settings] AutoRecord=1`
- 自动录音进行中时，复选框禁用（不允许中途取消）

## 四、数据流

```
WM_TIMER (ID=2, 1秒)
        │
        ▼
   IsWeChatInCall()
   EnumWindows → 遍历所有顶层窗口
        │
   GetClassNameW → 匹配 "ILinkAudioWnd" / "AudioWnd" / "ILinkVoipTrayWnd"
        │
        ▼
   ┌─ 找到 ≥1 个窗口 ──────────────────────────┐
   │                                             │
   ▼                                             ▼
  当前未在录音？                                通话窗口全部消失？
   ├─ 是 → 找微信PID → 开始录音                 ├─ 是 → 等待3秒确认 → 停止录音
   └─ 否（已在录）→ 什么都不做                   └─ 否 → 什么都不做
```

## 五、边界情况

| 场景 | 处理 |
|------|------|
| 微信在通话但 PID 暂未出现 | 每秒重试，最多 10 次，仍未找到 → 放弃本次 |
| 用户已在手动录其他软件 | 不触发自动录音 |
| 自动录音中，用户手动点停止 | 允许停止，但不会立即自动重开（需等下次通话触发） |
| 通话短暂断线（窗口闪现消失） | 3 秒冷却期，窗口持续消失 3 秒才判定通话结束 |
| 微信没在通话，自动模式开着 | 什么都不做，正常待机 |
| 首次启动还没有 INI | AutoRecord 默认 1（开启） |

## 六、测试计划

1. 打开 APP，看到"微信通话自动录音"复选框
2. 勾选状态，拨通微信电话 → APP 自动开始录音
3. 挂断电话 → APP 自动停止，文件正常保存
4. 取消勾选，拨通微信电话 → APP 不自动录音
5. 手动录着其他软件时，微信来电 → 不自动切过去
